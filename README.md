# Healthcare Cost & Expenditure Analysis

## Project Overview

This project analyzes healthcare expenditure and insurance cost data to identify high-cost patient patterns, demographic spending variations, and regional healthcare cost trends using Python and Tableau.

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Tableau Public
* Jupyter Notebook

## Dataset

Medical Cost Personal Dataset obtained from Kaggle containing healthcare insurance and medical expenditure information for 1,338 individuals.

## Project Workflow

* Data loading and preprocessing
* Exploratory data analysis
* Healthcare expenditure trend analysis
* Demographic cost analysis
* Data visualization
* Tableau dashboard development
* KPI-based reporting

## Key Analyses

The project evaluated:

* Smoker vs non-smoker healthcare costs
* Regional healthcare expenditure trends
* Age vs medical charges
* BMI and healthcare spending patterns
* Overall healthcare cost distributions

## Key Findings

* Smokers showed significantly higher medical charges compared to non-smokers
* Southeast region recorded the highest average healthcare costs
* Healthcare charges generally increased with age
* Average BMI exceeded 30, indicating obesity-related healthcare risk trends
* Healthcare expenditure distribution showed high-cost patient concentration patterns

## Visualizations

* Healthcare Charges Distribution
* Smoker vs Non-Smoker Cost Comparison
* Regional Healthcare Charges
* Age vs Medical Charges Scatter Plot
* Tableau Healthcare Cost Dashboard

## Tableau Dashboard

A Tableau dashboard was developed to present healthcare expenditure KPIs and demographic cost trends through interactive visual analytics.
